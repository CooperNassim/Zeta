# 智能交易系统

一个基于 React 的智能交易管理系统，集成了心理测试、交易策略、风险模型等智能评估功能。

## 功能特性

### 1. 首页
- 炫酷的机器人动画效果
- 实时统计数据显示
- 核心功能展示

### 2. 每日功课
- 维护全球资产价格指数
- 支持股票、加密货币、外汇等类型
- 实时涨跌显示

### 3. 心理测试
- 灵活设定测试指标
- 自定义评分标准和权重
- 心理状态评估和记录

### 4. 交易策略
- 买入和卖出策略管理
- 多条件评估机制
- 策略评分系统

### 5. 风险模型
- 多种风险管控模型
- 仓位计算模拟器
- 自动计算建议买入数量

### 6. 预约单
- 智能评估流程
- 心理测试 → 策略评估 → 风险控制
- 支持执行和取消

### 7. 账单明细
- 自动记录交易流水
- 手动入账/出账
- 资金变动统计

### 8. 交易记录
- 买入卖出记录
- 评分统计
- 盈亏分析

## 技术栈

- **React 18** - 前端框架
- **Vite** - 构建工具
- **Tailwind CSS** - 样式框架
- **Framer Motion** - 动画库
- **Zustand** - 状态管理
- **Lucide React** - 图标库
- **date-fns** - 日期处理

## 使用说明

### 1. 开始使用

1. 首次使用时，建议先完成心理测试
2. 配置交易策略和风险模型
3. 维护每日功课中的资产价格
4. 创建预约单时系统会自动评估

### 2. 创建买入预约单流程

1. 点击"买入预约"按钮
2. 心理测试评估（必须通过）
3. 选择交易策略并评估
4. 选择风险模型并计算仓位
5. 填写订单信息
6. 创建成功后可在订单列表中执行

### 3. 创建卖出预约单流程

1. 点击"卖出预约"按钮
2. 心理测试评估（必须通过）
3. 选择卖出策略并评估
4. 填写订单信息
5. 创建成功后可在订单列表中执行

### 4. 记录交易盈亏

在"交易记录"页面，对于买入交易可以在卖出时填写盈亏金额，系统会自动计算统计数据。

## 数据存储

所有数据使用浏览器本地存储（localStorage）持久化保存，刷新页面不会丢失数据。

## 特色功能

### 智能评估系统
- 心理测试评估：确保交易时情绪稳定
- 策略评估：根据预设条件自动评分
- 风险控制：科学计算仓位和止损

### 可视化数据
- 实时统计图表
- 评分进度条
- 盈亏趋势分析

### 响应式设计
- 适配不同屏幕尺寸
- 优雅的动画效果
- 玻璃态设计风格

## 注意事项

1. 本系统仅为模拟交易系统，不构成投资建议
2. 数据仅保存在本地浏览器中
3. 清除浏览器数据会丢失所有记录
4. 建议定期导出备份重要数据
